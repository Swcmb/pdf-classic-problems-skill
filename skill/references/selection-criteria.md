# 选题标准

> 本文档定义经典题筛选的三大标准、评分规则、筛选流程与降级策略。
> LLM 在阶段 4（经典题筛选）参照本文件执行。

## 1. 三大标准与权重

| 标准 | 权重 | 核心问题 |
|------|------|----------|
| 核心概念代表性 | 40% | 是否直接考察核心概念？是否综合运用多个知识点？ |
| 方法论启发性 | 35% | 是否展示重要解题技巧？是否有多种解法？ |
| 典型性与易错性 | 25% | 是否为常见考点？是否包含常见错误？ |

## 2. 评分规则

### 2.1 评分维度

每项标准评分 1-5 分：

| 分值 | 含义 |
|------|------|
| 5 | 极具代表性/极具启发性/极易出错 |
| 4 | 较具代表性/较具启发性/较易出错 |
| 3 | 一般代表性/一般启发性/一般典型性 |
| 2 | 代表性较弱/启发性较弱/典型性较弱 |
| 1 | 几乎无代表性/无启发性/无典型性 |

### 2.2 入选候选池

每项标准评分 ≥ 3 分的题目入选候选池。

### 2.3 综合评分公式

```
总分 = 核心概念 × 0.40 + 方法论 × 0.35 + 典型性 × 0.25
```

### 2.4 入选阈值

总分 ≥ 3.5 的题目方可入选经典题列表。

## 3. 筛选流程

1. 对所有识别到的题目按三大标准评分
2. 计算综合评分
3. 按总分降序排列
4. 选取前 N 题（默认 10 题，用户可配置范围为 8-12 题）
5. 确保覆盖各主要章节（每章至少 1 题，如有）
6. 若同章节多题高分，取最高分者

## 4. 数量不足时的降级策略

| 总题数 | 策略 |
|--------|------|
| < 8 | 选取总题数的 50-80%，提示用户 |
| < 3 | 提示用户 PDF 可能不含足够题目 |
| = 0 | 转入异常处理（题目识别为空） |
