# 题目编码系统

> 本文档定义章节识别、题号识别、题目边界判定与编码生成规则。
> LLM 在阶段 3（题目识别）参照本文件执行。

## 1. 章节标题识别

| 模式 | 正则 | 示例 |
|------|------|------|
| 中文数字章节 | `第[一二三四五六七八九十百]+章` | 第十章 随机事件与概率 |
| 阿拉伯数字章节 | `第\d+章` 或 `\d+\.\d*(\.\d+)*` | 第10章 或 10.1.1 |

**识别规则**：
- 章节标题通常独占一行，前后有空行
- 阿拉伯数字章节号支持多级（如 `10.1.1` 表示第 10 章第 1 节第 1 小节）
- 章节标题后可能紧跟章节名称（如"第十章 随机事件与概率"）

## 2. 题号识别

| 题型 | 正则模式 | 编码前缀 | 上下文约束 |
|------|----------|----------|------------|
| 例题 | `例\d+`、`例\d+\.\d+`、`Example \d+` | EX | 题号位于行首或段首 |
| 习题 | `习题\d+`、行首`\d+\)`、行首`\(\d+\)` | EXS | 题号位于行首，且后接非标点正文 |
| 定理证明 | `定理\d+.*证明` | T | 需包含"证明"关键词 |
| 定义推导 | `定义\d+.*推导` | D | 需包含"推导"关键词 |

### 2.1 误匹配过滤规则

以下规则用于排除正文中对题号的引用，避免误识别：

1. **位置约束**：题号必须位于行首或前导空白后（排除正文中引用如"在 3) 的情况下"）
2. **内容约束**：题号后必须紧跟题目正文内容（排除"(2) 见下文"等引用语）
3. **上下文约束**：习题题号需结合当前章节上下文判定（确认位于习题区域而非正文段落）

## 3. 题目边界判定

### 3.1 起点与终点

- **起点**：题号行或章节标题后的第一个问题
- **终点**：下一题号行、下一章节标题、或解答结束标记（如"解："后的完整解答）

### 3.2 跨页判定规则

满足以下任一条件即判定为跨页题目：

1. 当前页末尾题号后内容未出现下一题号或章节标题
2. 当前页末行非自然段结束标志（未以句号、问号、分号结尾）
3. 下一页首行内容与上一题题干语义连续（非新题号、非章节标题）

### 3.3 跨页处理

- 判定为跨页时，合并前后页内容
- 拼接处去除分页符

### 3.4 公式完整性

- 确保公式未被截断
- 检查 `$`、`$$` 配对
- 若不配对则向下一页延伸至配对完成

## 4. 题目编码生成

### 4.1 编码格式

```
章节号-题型序号
```

### 4.2 示例

```
10.1.1-EX1    = 第 10.1.1 节的第 1 道例题
10.1.1-EX2    = 第 10.1.1 节的第 2 道例题
10.2-EXS3     = 第 10.2 节的第 3 道习题
10.3-T1       = 第 10.3 节的第 1 个定理证明
```

### 4.3 编码规则

- **多级章节**：取最细粒度章节号（如 `10.1.1` 而非 `10.1`）
- **同章节同类型递增序号**：同一章节同一题型内序号从 1 递增
- **题型前缀**：EX（例题）、EXS（习题）、T（定理证明）、D（定义推导）
