# 输出格式规范

> 本文档定义两个输出 Markdown 文件的格式模板与公式格式要求。
> LLM 在阶段 5（详解生成与输出）参照本文件执行。

## 1. 经典题列表.md

### 1.1 格式模板

```markdown
# 经典题列表

## {章节号} {章节名称}
- {题号1}
- {题号2}

## {下一章节号} {下一章节名称}
- {题号3}
```

### 1.2 规则

- 按章节号升序排列
- 同章节内按题号顺序排列
- 仅含题号，不含题目内容

## 2. 经典题详解与拓展.md

### 2.1 格式模板

```markdown
# 经典题详解与拓展

---

## {题号}

### 题目
{原题内容，保留公式与格式}

### 解答
{完整解题过程，逐步推导}

### 分析
{解题思路说明，关键步骤解释}

### 反思与拓展
**方法总结**：{该方法论的核心要点}

**拓展题1**：
{拓展题内容}

<details>
<summary>拓展题解答</summary>
{拓展题解答}
</details>
```

### 2.2 规则

- 每题包含四部分：题目、解答、分析、反思与拓展
- 题间使用 `---` 分隔线
- 拓展题解答使用 `<details>` 标签折叠，默认折叠状态
- 每题至少提供 1 道拓展题

## 3. 公式格式要求

| 场景 | 格式 | 示例 |
|------|------|------|
| 行内公式 | `$...$` | $P(A \cup B)$ |
| 行间公式 | `$$...$$` | $$P(A|B) = \frac{P(AB)}{P(B)}$$ |
| 多行公式 | `\begin{align}...\end{align}` | 对齐推导步骤 |
| 分段函数 | `\begin{cases}...\end{cases}` | 定义域分段 |
| 分隔线 | 题间 `---` | 视觉分隔 |
| 拓展题解答 | `<details>` 折叠 | 默认折叠 |

### 3.1 公式书写规范

- 行内公式与前后文字之间保留一个空格
- 行间公式前后各空一行
- 多行公式使用 `\\` 换行，`&` 对齐
- 确保所有 `$` 和 `$$` 配对闭合
- 特殊符号使用 LaTeX 命令（如 `\cup`、`\cap`、`\frac`、`\sum`、`\int`）
